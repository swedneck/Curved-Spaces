Para dirigir
arrastra con el ratón
Para cambiar tu velocidad
presiona la tecla ( ↑ ) para ir más deprisa o la tecla ( ↓ ) para ir más despacio
Podrías tener que hacer click sobre la ventana principal para activar la
Para pausar
presiona la barra espaciadora
Podrías tener que hacer click sobre la ventana principal para activar la
Para trasladar horizontalmente o verticalmente
mantén la tecla Shift pulsada mientras arrastras con el ratón
Para trasladar adelante o atrás
mantén las teclas Shift y Control pulsadas mientras arrastras verticalmente con el ratón
Para rolar
rueda con dos dedos en el trackpad
Se no tienes trackpad, mantén la tecla Control pulsada mientras arrastras horizontalmente con el ratón
Para abrir o cerrar la abertura
pellizca con dos dedos en el trackpad
Se no tienes trackpad, presiona la tecla ( ← ) para contraer la abertura o la tecla ( → ) para expandir la abertura
