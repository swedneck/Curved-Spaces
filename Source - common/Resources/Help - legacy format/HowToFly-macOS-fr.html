Pour piloter
déplacer la souris dans la direction désirée
Pour changer de vitesse
la flèche vers le haut ( ↑ ) permet d’accélérer, la flèche vers le bas ( ↓ ) de décélérer
Vous devrez peut-être cliquer sur la fenêtre principale pour l’activer
Pour s’arrêter
appuyez sur la barre espace
Vous devrez peut-être cliquer sur la fenêtre principale pour l’activer
Pour se déplacer latéralement, vers le haut ou vers le bas
appuyez sur la touche Shift tout en déplaçant la souris
Pour se déplacer en avant ou en arrière
appuyez simultanément sur les touches Shift et Control tout en déplaçant la souris verticalement
Pour modifier l’angle de roulis
faites une rotation avec deux doigts sur le pavé tactile
Faute de pavé tactile, appuyez sur la touche Control tout en déplaçant la souris horizontalement
Pour modifier l’ouverture
pincez avec deux doigts sur le pavé tactile
Faute de pavé tactile, appuyez sur la flèche à gauche ( ← ) pour contracter l’ouverture ou sur la flèche à droite ( → ) pour la dilater
