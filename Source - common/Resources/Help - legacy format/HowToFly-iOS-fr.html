Pour piloter
inclinez votre iPhone ou votre iPad vers la direction désirée
ou faites glisser votre doigt
Pour changer de vitesse
utilisez le curseur
Pour s’arrêter
tapez d’un doigt
Pour se déplacer latéralement, vers le haut ou vers le bas
faites glisser deux doigts
Pour modifier l’angle de roulis
faites une rotation avec deux doigts
Pour modifier l’ouverture
pincez avec deux doigts
