Para guiar
arrasta com o rato
Para mudar a velocidade
pressiona a tecla cima ( ↑ ) para ires mais rápido ou a tecla baixo ( ↓ ) para ires mais devagar
Talvez tenhas de clicar primeiro na janela principal para a ativar
Para pausar
pressiona a tecla espaço
Talvez tenhas de clicar primeiro na janela principal para a ativar
Para deslocar lateralmente, para cima ou para baixo
mantém pressionada a tecla Shift enquanto arrastas com o rato
Para deslocar para a frente ou para trás
mantém pressionada ambas as teclas Shift e Control enquanto arrastas verticalmente com o rato
Para rolar
roda com dois dedos no trackpad
Se não tiveres um trackpad mantém pressionada a tecla Control enquanto arrastas horizontalmente com o rato
Para abrir ou fechar a abertura
aproxima ou afasta dois dedos no trackpad
Se não tiveres um trackpad pressiona a tecla esquerda ( ← ) para contrair a abertura ou a tecla direita ( → ) para expandir a abertura
