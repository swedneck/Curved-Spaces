To steer
drag with the mouse
To change your speed
press the up arrow key ( ↑ ) to go faster or the down arrow key ( ↓ ) to go slower
You might first need to click on the main window to make it active
To pause
press the spacebar
You might first need to click on the main window to make it active
To slide sideways, up or down
hold the Shift key down while dragging with the mouse
To slide forwards or backwards
hold the Shift and Control keys down while dragging vertically with the mouse
To roll
rotate with two fingers on the trackpad
If no trackpad is available, hold the Control key down while dragging horizontally with the mouse
To open or close the aperture
pinch with two fingers on the trackpad
If no trackpad is available, press the left arrow key ( ← ) to contract the aperture or the right arrow key ( → ) to expand the aperture
